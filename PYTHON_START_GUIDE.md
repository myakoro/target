# 🐍 Python版 TARGET自動化 - クイックスタート

## ✅ 準備

1. **Pythonの確認**:
   ```powershell
   python --version
   ```

2. **ライブラリのインストール**:
   ```powershell
   pip install pyautogui pillow opencv-python
   ```

## 🚀 実行手順

1. **TARGETを起動し、最大化**してください。

2. **「成績」ボタン → 「開催名」タブ** をクリックし、開催日一覧を表示させてください。

3. **スクリプト実行**:
   ```powershell
   cd C:\Users\takuy\Desktop\投資競馬アプリ系\target系\scripts
   python target_automation.py
   ```

4. **素早くTARGETウィンドウをクリック**して最前面にしてください（5秒の猶予があります）。

## ⚠️ 注意事項

- **マウスを動かさないでください**: 実行中はマウスが勝手に動きます。
- **緊急停止**: マウスを画面の**左上隅**に素早く移動させると、プログラムが強制終了します。
- **座標ズレ**: もしクリック位置がずれる場合は、`target_automation.py` の `COORDS` 値を調整してください。

## 📝 処理内容

1. 1番目の開催日（一番上の行）をダブルクリック
2. 1R～12Rを順番に処理
   - ダブルクリックしてレース詳細へ
   - オッズボタン → 単複時系列タブ → JVオッズ取得
   - 30秒待機
   - CSV出力 (Alt+F -> T -> C)
   - 閉じる (Alt+F4 x 2)
   - 次のレースへ
